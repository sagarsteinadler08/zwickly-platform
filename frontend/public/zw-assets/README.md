# Zwickly Assets Directory

This directory is for hero images and event images used in the Zwickly app.

## Usage

- Drop hero & event images here
- Filenames must match existing imports in components
- If you replace files, keep original filenames or update the component imports

## Notes

- hero image: swap file at `public/zw-assets/hero-1.jpg` to update hero
- Event images are typically referenced from the `assets` directory or external URLs
